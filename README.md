# interview
This repo is intended to store the code related to an specific interview task.

## Go Development Demo Task

# Objective​:
To measure data structure knowledge and implementation abilities of the candidate.
Please implement stack and queue data structures using a go lang.

# Deliverables​:
Please provide a simple document which explains how you would implement stack and queue
data structures using go with a clean code sample.

# Bonus​:
Which go library would you use if you did not want to do it yourself ?
